# open-app

Electron + React + TypeScript monorepo (pnpm workspaces).

## Quick start

```bash
pnpm install
pnpm dev
```

This starts the Vite dev server and Electron.

## Dev

- `pnpm dev` — start desktop app
- `pnpm test` — run unit tests
